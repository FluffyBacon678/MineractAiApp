Behaviour Tuning
Internal Monologue
Background self-reflection — Bud checks if he's on track
Enable internal monologue
5 / 10
When enabled, Bud quietly reflects on his current state and task. If something is wrong he emits an ADJUST directive logged in Activity Log. Neuroticism 10 = fires almost every tick; 1 = rarely.
Conversation Memory
How many exchanges Bud remembers mid-conversation
20 turns
Each "turn" is one player message + one Bud reply. Quick tier gets 6 turns, Standard gets 12, Deep gets all. When the buffer is full, older turns are summarised into a single line so context is never completely lost.
Planning & Auto-Execution
What happens when Bud generates a multi-step plan
Auto-execute plan steps
Confirm each step in chat
When Bud produces a numbered plan (e.g. 1. Go to farm 2. Harvest), auto-execute will run each step in sequence. Confirm mode asks "ok?" in chat before each step. You can also just let Bud say the plan without executing.
Social Energy
Controls how Bud's social battery charges and drains
Enable social battery
8 pts / exchange
5 pts / min
At 100% Bud initiates chat freely. Below 30% he drops a model tier (shorter answers). Below 10% only quick-tier responses. Recharges after 30 s of quiet.
